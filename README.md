# 🔢 Bin2Dec  
### Binary to Decimal Converter

Projeto simples de conversor de número binário para decimal.

## 🚀 Funcionalidades
- Conversão de binário para decimal
- Validação de entrada (apenas 0 e 1)
- Mensagens de erro e sucesso
- Interface simples e responsiva

## 💻 Tecnologias
- HTML
- CSS
- JavaScript

## 🎯 Aprendizados
- Manipulação do DOM
- Eventos em JavaScript
- Validação de inputs
- Estilização de UI

## 📌 Status do projeto

✔ Finalizado  
✔ Publicado  
✔ Funcional

## 🚀 Possíveis melhorias

- Histórico de conversões  
- Modo escuro  
- Animação no resultado  

## ▶️ Como usar

1. Digite um número binário (ex: 1010)
2. Clique em “Converter”
3. Veja o resultado em decimal
